# Online Food Ordering System

## Project Description
This project is used to order food online from restaurants.

## Features
- Customer Login
- Restaurant Management
- Food Ordering
- Online Payment
- Order Tracking

## Technologies Used
- Java
- MySQL
- GitHub

## Setup Instructions
1. Install Java
2. Install MySQL
3. Clone Repository
4. Open in IDE
5. Run the project
